## ✔️ Commit Message Convention

### 1. commit struct

- `[타입]: 제목 (필수)`

- `[본문]: 변경 내용 (옵션)`

> 💡 [feat]: 로그인 기능 추가 #1(이슈 번호 작성) <br /> - 변경 상세 내용 <br /> - ...



### 📒 Branch Strategy

배포 브랜치는 통합테스트까지 완료한 상태로 유지되어야 하며, 배포 시 사용될 브랜치 입니다.

- 배포 브랜치 : 'main'

- 개발 브랜치 : 'dev'

```
main
dev
```
